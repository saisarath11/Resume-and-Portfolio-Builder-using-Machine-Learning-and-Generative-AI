# Resume-and-Portfolio-Builder-using-Machine-Learning-and-Generative-AI:
An AI-powered web application that automatically generates professional resumes and portfolio websites based on user input. The system leverages Machine Learning and Generative AI (LLMs) to create optimized, ATS-friendly resumes tailored to specific job roles.

 # Features:
 
 Automated Resume Generation
 Portfolio Website Generation
 AI-Based Content Enhancement
 Job Role–Specific Customization
 ATS Optimization
 Editable & Downloadable Resume (PDF)
 Smart Skill Suggestions

# How It Works:

User enters personal, academic, and professional details.
The ML model analyzes skills and job role preferences.
Generetive AI enhances content professionally.
The system generates:
A structured resume
A customizable portfolio webpage
